# Astronetics

Web-часть проекта, разработанного в рамках дисциплины «IT-технологии в исследовании космоса» (Проектно-исследовательская студия по дисциплине «Россия в мире: глобальные вызовы и тренды»)

Разработано и проверено с использованием:
- PHP 8.2.5
- MariaDB 10.6
- Nginx 1.18

Код мобильного приложения расположен в репозитории: <https://github.com/samrzhevsky/astronetics>
